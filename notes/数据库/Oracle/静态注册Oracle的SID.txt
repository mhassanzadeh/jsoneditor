====== 静态注册Oracle的SID ======
Created Friday 04 June 2010

修改listener.ora里面


SID_LIST_LISTENER =
  (SID_LIST =
    (SID_DESC =
      (GLOBAL_DBNAME = powerwork)
      (ORACLE_HOME = F:\Oracle\product\11.1.0\db_1)
      (SID_NAME = POWERWORK)
    )
  )

LISTENER =
  (DESCRIPTION_LIST =
    (DESCRIPTION =
      (ADDRESS = (PROTOCOL = TCP)(HOST = hydt09)(PORT = 1521))
    )
  )

然后重启监听 lsnrctl reload
