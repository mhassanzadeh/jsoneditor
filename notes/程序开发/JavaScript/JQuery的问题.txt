====== JQuery的问题 ======
Created Friday 04 June 2010

如果页面不刷新，一直使用.click绑定到同一个对象，那么这些click事件就会累加。

$(obj).click(alert(1));
$(obj).click(alert(2));
$(obj).click(alert(3));

点 obj 之后，就会出来 3 个 alert 。所以绑定前先把原来的click解绑，再绑定新动作。

$(obj).unbind('click');
$(obj).click(alert(1));

或者在click里面定义，执行完之后解绑。

$(obj).click(function() {
 alert(1);
 $(obj).unbind('click');
});
